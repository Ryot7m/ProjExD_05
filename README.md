# こうかとん無双・極 (仮)

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
こうかとんが敵機を撃ち落とすゲーム

## ゲームの実装
### 共通基本機能
* 背景画像とこうかとんの描画

### 担当追加機能
* スタート画面の追加
* こうかとんのサイズを縮小する
* 敵機の巨大化
* 入力キーでbeamの効力を変更する(基のプログラムからbeamの速さ等改良)、ビームの命中に応じてレベルを上げる(担当　久保田)
* 防御壁をこうかとんの前後に追加する
